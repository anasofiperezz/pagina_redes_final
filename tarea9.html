¿Qué hace la etiqueta input de HTML y como se utilizan?

La etiqueta input de HTML permite la creación de un campo de entrada de datos en un formulario en línea. El atributo type de la etiqueta input determina el tipo de campo que se va a mostrar, como texto, contraseña, botón, radio, casilla de verificación, etc.

¿Qué es el lenguaje JavaScript?

JavaScript es un lenguaje de programación interpretado utilizado para crear interactividad en las páginas web. Es un lenguaje de alto nivel, orientado a objetos y con una sintaxis similar a C.

¿Para qué sirve el leguaje JS?

JavaScript se utiliza para agregar interactividad a las páginas web, como validar formularios, crear efectos visuales y dinámicos, crear animaciones, realizar solicitudes de red, etc.

¿A qué se refiere el DOM (Document Object model) en el desarrollo Web?

El DOM es una representación en memoria de un documento HTML que permite interactuar con los elementos de la página web mediante JavaScript. Permite la modificación y manipulación dinámica de la estructura, contenido y estilo de la página.

¿Cuál es la diferencia entre los archivos HTML, CSS y JS?

HTML es un lenguaje de marcado utilizado para crear la estructura y contenido de una página web. CSS es un lenguaje de estilos utilizado para darle estilo y presentación a la página web. JavaScript es un lenguaje de programación utilizado para agregar interactividad y funcionalidad a la página web.

¿Para qué sirve la etiqueta ?script?>?

La etiqueta ?script?> se utiliza para insertar código JavaScript en una página HTML.

¿En qué consiste los 3 métodos para cambiar elementos tipo Tag Name, ID y Class Name con JS?

Los métodos para cambiar elementos en JavaScript son:

  • getElementById: permite obtener un elemento mediante su identificador (ID).
  • getElementsByClassName: permite obtener un arreglo de elementos mediante su clase.
  • getElementsByTagName: permite obtener un arreglo de elementos mediante su nombre de etiqueta.

    ¿Cuál es la sintaxis de la programación en JS?

    La sintaxis básica de JS es similar a la de otros lenguajes de programación. Se utilizan variables, operadores, condicionales, ciclos, funciones, etc. La sintaxis se divide en sentencias, que se separan por punto y coma.

    ¿Cómo se declaran variables en JS?

    Las variables en JS se declaran usando la palabra clave "var", "let" o "const", seguida del nombre de la variable y opcionalmente un valor inicial. Por ejemplo:

  • var num = 10;
  • let name = "Juan";
  • const pi = 3.14;

    ¿Cuáles ciclos de programación están disponibles en JS?

    Los ciclos disponibles en JS son "for", "while" y "do-while".

    ¿Cuáles funciones de condicional están disponibles en JS?

    Las funciones de condicional disponibles en JS son "if", "else if", "else", "switch" y "ternary operator".
